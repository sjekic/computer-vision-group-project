# Data Directory

Expected layout:

```text
raw/<location>/*.jpg
processed/<location>/*.jpg
query/<location>/*.jpg
```

Use `raw/` for database/reference captures and `query/` for real held-out
evaluation images. `processed/` is generated by `src/preprocess.py`.

The `.gitignore` keeps image files out of git by default. For submission, share
the full dataset or a representative subset through the course-approved channel
and document it in `docs/dataset_card.md`.
